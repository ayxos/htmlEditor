Html Editor is a simple tool to create your html code, mainly focus on interpreters who are translating a web page but also appropriate for Html content editors.

    Features list:
  • Timer to check your translation speed
  • Word counter
  • Calc to get your revenue per word
  • Download link to get the final html code
